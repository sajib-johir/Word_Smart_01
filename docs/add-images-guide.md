# Add Images Guide

1. Put new images inside `images/wordsmart-1/<LETTER>/`.
2. You can also use pack subfolders such as `pack-01`, `pack-02`.
3. Keep filenames in the preferred format: `Number. Word.webp`.
4. Run `python tools/check_images.py`.
5. Run `python tools/generate_words.py`.
6. Open `index.html` and test search, navigation, and progress.
